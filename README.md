# Hi, I'm Mahbube Bejam

I build software where engineering meets real industrial operations.

My background combines full-stack software engineering, AI-driven application development, and hands-on experience within pharmaceutical manufacturing environments.

Working directly alongside production and quality teams has given me a practical understanding of GMP workflows, visual inspection, batch documentation, deviation management, traceability, and the operational challenges that digital solutions are expected to solve.

Today, my focus is designing intelligent software that simplifies complex processes, supports decision-making, and helps organizations adopt AI in practical, human-centered ways.
